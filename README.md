# IA_No_Ensino_de_Matematica
Repositório do desafio em IA no ensino de Matemática: transforma o aprendizado em prática. Com ChatGPT e Copilot, aborda planejamento, materiais, internacionalização e adaptação para 2 perfis de alunos. Focado em autoria, ética e proteção de dados, é ideal para treinar decisões e fluência em IA educativa.
